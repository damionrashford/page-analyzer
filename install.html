Installation Instructions

  1. Make sure your bookmarks bar is visible (Ctrl+Shift+B on Windows/Linux or Cmd+Shift+B on Mac)
  2. Drag the button above to your bookmarks bar
  3. Done! Click it on any webpage to analyze it

Manual Installation

If drag-and-drop doesn't work, right-click the button above and select "Copy link address", then:

  1. Right-click your bookmarks bar
  2. Select "Add page" or "Add bookmark"
  3. Name it "Page Analyzer"
  4. Paste the copied code into the URL field
  5. Click "Save"
